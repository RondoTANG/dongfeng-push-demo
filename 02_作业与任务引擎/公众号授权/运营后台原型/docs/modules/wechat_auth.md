# 公众号管理模块 PRD

## 1. 功能概述
展示已授权的公众号矩阵，提供扫码弹窗，以及已群发文章列表。

## 2. 逻辑说明与图钉批注

### 📍 图钉批注：wechat_auth_status
- **功能名称**：公众号授权状态监控
- **功能定义**：展示当前绑定公众号的 Token 存活性。
- **交互说明**：正常监听为绿色。如 Token 失效或被解绑，标记为红色“已失效”，并高亮引导重新扫码的入口。
- **接口信息**：调用微信开放平台 `api_get_authorizer_info` 获取授权方基础信息。
- **关联字段**：
  - `authorizer_appid`: 授权公众号的 AppID
  - `authorizer_refresh_token`: 刷新令牌
  - `status`: 根据 token 刷新接口返回值判断，若返回 `61023` 或 `40001` 等错误码，则置为“已失效”。

### 📍 图钉批注：wechat_scan_auth
- **功能名称**：新公众号扫码授权
- **功能定义**：接入新的公众号矩阵，通过生成二维码引导管理员扫码。
- **交互说明**：扫码成功后的回调需二次确认是否包含“群发与发布权限”及“消息管理权限”，若缺项则前端抛出 Toast 弹窗报错。
- **接口信息**：调用微信第三方平台 `api_create_preauthcode` 获取预授权码，跳转至开放平台扫码页。授权成功后调用 `api_query_auth`。
- **关联字段**：
  - `pre_auth_code`: 预授权码，用于拼接扫码链接。
  - `func_info`: 授权成功后返回的权限集，必须包含 `id: 1` (消息管理) 与 `id: 15` (图文素材/群发管理)。

### 📍 图钉批注：article_link_status
- **功能名称**：已群发文章的作业关联状态
- **功能定义**：展示所有监听公众号的推文历史，并指示是否已经生成了推送作业。
- **交互说明**：分为“未关联”和“已关联”。未关联时主色调为原色，显示“生成推送作业”主操作；已关联时，整个卡片降低透明度至 80%，按钮变为次级的“查看作业看板”。
- **接口信息**：调用微信开放平台 `freepublish/batchget` (获取成功发布列表)。
- **关联字段**：
  - `article_id`: 图文文章唯一 ID
  - `title`: 图文消息的标题
  - `content_url`: 图文消息的链接，用于作业下发时配置。
  - 业务侧自维护字段：`job_id`，若不为空则表示该文章已绑定作业。
