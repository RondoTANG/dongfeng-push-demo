# 公众号授权运营后台原型总纲 PRD

## 1. 业务背景
为了实现“护卫军”体系中的自动化任务分发与数据回收，运营后台需要接入各大主机的公众号矩阵。通过扫码授权，系统能够监听矩阵下的推文，快速生成推广作业。

## 2. 核心模块与受众
- **公众号管理模块**：针对管理员。用于查看授权存活性、接入新账号（扫码授权）、查看所有已授权矩阵的历史推文。
- **作业管理模块**：针对运营。用于向导化创建作业，并从数据库内嵌选取公众号文章关联任务，最后预估扣减品牌额度池后发布。

## 3. Demo 验证目标
- **业务验证**：验证全量交互闭环。如扫码弹窗的条件是否合理、文章池状态切换逻辑是否清晰、新建作业四步向导的级联关系是否无断点。
- **技术解耦**：本页面全部使用 Vanilla JS/CSS 构建，没有任何复杂的框架代码。全部逻辑使用 Markdown 注释与 `data-key` 关联，便于研发直接在原型图钉上查看接口与取值逻辑。
